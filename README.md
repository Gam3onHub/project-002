# SaaS Homepage

A clean and modern SaaS landing page designed to showcase a productivity dashboard platform.

## Features

- Fully responsive layout (mobile, tablet, desktop)
- Sticky header with glassmorphism effect on scroll
- Interactive feature cards with 3D flip animation
- Animated statistics counter on scroll
- Smooth mobile hamburger menu
- Professional dark theme with accent color

## Technologies Used

- HTML5
- CSS3 (Flexbox, CSS Grid, Clamp function, Custom Properties)
- Vanilla JavaScript
- AOS (Animate On Scroll Library)
